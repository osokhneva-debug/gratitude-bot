# Дневник Благодарностей

## Что это
Telegram-бот для ежедневной практики благодарности. Пользователь записывает 10 вещей, за которые благодарен сегодня, и формирует красивый архив записей, который можно листать как личный дневник.

## Как работает
1. Бот присылает напоминание вечером: "За что ты благодарен сегодня?"
2. Пользователь вводит 10 благодарностей (по одной или списком)
3. Бот оформляет запись в красивую карточку с датой и сохраняет
4. Пользователь листает архив кнопками ← →, читая прошлые записи

## Core-фича
**Красивые карточки записей** с пагинацией — каждый день оформлен как страница дневника, которую можно листать вперёд-назад

## Стек
- **Python 3.11+** — язык
- **aiogram 3.x** — фреймворк для Telegram-бота
- **SQLite** — хранение записей (простой файл, не нужен сервер)
- **APScheduler** — напоминания по расписанию

## Установка

```bash
pip install aiogram apscheduler aiosqlite
```
